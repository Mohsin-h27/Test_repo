1""" 

2SCIM Service Provider Configuration Module 

3 

4This module provides functionality for managing and retrieving the SCIM (System for Cross-domain 

5Identity Management) service provider configuration in the Workday Strategic Sourcing system. 

6 

7The module interfaces with the simulation database to provide access to the service provider 

8configuration, which details the SCIM specification features and capabilities supported by the 

9system, including authentication schemes, supported operations, and bulk configuration. 

10 

11Functions: 

12 get: Retrieves the complete service provider configuration 

13""" 

14 

15from typing import Dict, Any, List 

16from .SimulationEngine import db 

17 

18def get() -> Dict[str, Any]: 

19 """ 

20 Retrieves the complete SCIM service provider configuration. 

21 

22 This function implements the SCIM service provider configuration endpoint as specified 

23 in RFC 7643, section 5. It returns detailed information about the SCIM features and 

24 capabilities supported by the system. 

25 

26 Returns: 

27 Dict[str, Any]: A dictionary containing the service provider configuration details. 

28 The dictionary includes the following key-value pairs: 

29 - "schemas" (List[str]): List of SCIM schemas used in the configuration 

30 - "documentationUri" (str): URI of the service provider's documentation 

31 - "patch" (Dict[str, bool]): Configuration for PATCH operations 

32 - "supported" (bool): Whether PATCH operations are supported 

33 - "bulk" (Dict[str, Any]): Configuration for bulk operations 

34 - "supported" (bool): Whether bulk operations are supported 

35 - "maxOperations" (int): Maximum number of operations in a bulk request 

36 - "maxPayloadSize" (int): Maximum payload size in bytes 

37 - "filter" (Dict[str, bool]): Configuration for filtering 

38 - "supported" (bool): Whether filtering is supported 

39 - "maxResults" (int): Maximum number of results returned 

40 - "changePassword" (Dict[str, bool]): Configuration for password changes 

41 - "supported" (bool): Whether password changes are supported 

42 - "sort" (Dict[str, bool]): Configuration for sorting 

43 - "supported" (bool): Whether sorting is supported 

44 - "etag" (Dict[str, bool]): Configuration for ETags 

45 - "supported" (bool): Whether ETags are supported 

46 - "authenticationSchemes" (List[Dict[str, Any]]): List of supported authentication schemes 

47 Each scheme dictionary contains: 

48 - "type" (str): The authentication scheme type 

49 - "name" (str): The name of the authentication scheme 

50 - "description" (str): A description of the authentication scheme 

51 - "specUri" (str): URI to the authentication scheme specification 

52 - "documentationUri" (str): URI to the authentication scheme documentation 

53 - "meta" (Dict[str, Any]): Configuration metadata 

54 - "location" (str): The configuration's URI 

55 - "resourceType" (str): The type of resource (ServiceProviderConfig) 

56 - "created" (str): Creation timestamp 

57 - "lastModified" (str): Last modification timestamp 

58 - "version" (str): The configuration version 

59 """ 

60 return db.DB["scim"]["service_provider_config"]

16 """ 

17 Retrieve information about all supported SCIM schemas. 

18 

19 This endpoint implements the SCIM schema discovery mechanism as specified in RFC 7644, 

20 section 3.4. It returns a list of all available schemas that can be used for 

21 validating and processing SCIM resources. 

22 

23 Returns: 

24 List[Dict[str, Any]]: A list of dictionaries, where each dictionary represents a SCIM schema. 

25 Each schema dictionary contains the following key-value pairs: 

26 - "id" (str): The schema's unique identifier 

27 - "name" (str): The schema's display name 

28 - "description" (str): A human-readable description of the schema 

29 - "attributes" (List[Dict[str, Any]]): List of attribute definitions, where each attribute 

30 contains: 

31 - "name" (str): The attribute name 

32 - "type" (str): The attribute data type 

33 - "multiValued" (bool): Whether the attribute can have multiple values 

34 - "description" (str): A human-readable description of the attribute 

35 - "required" (bool): Whether the attribute is required 

36 - "caseExact" (bool): Whether the attribute value is case-sensitive 

37 - "mutability" (str): The attribute's mutability (readOnly, readWrite, immutable, writeOnly) 

38 - "returned" (str): When the attribute is returned (always, never, default, request) 

39 - "uniqueness" (str): The attribute's uniqueness constraint (none, server, global) 

40 - "meta" (Dict[str, Any]): Schema metadata including: 

41 - "location" (str): The schema's URI 

42 - "resourceType" (str): The type of resource (Schema) 

43 - "created" (str): Creation timestamp 

44 - "lastModified" (str): Last modification timestamp 

45 - "version" (str): The schema version 

46 """ 

47 return db.DB["scim"]["schemas"]
